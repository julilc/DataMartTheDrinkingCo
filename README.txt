Este proyecto es un trabajo práctico final para la asignatura Bases de datos II correspondiente a la carrera Tec. Universitaria en Inteligencia Artificial.
Dicho trabajo tiene el objetivo de realizar un DataMart de Ventas para una compañía llamada "The Drinking Company", y luego la explotación del mismo.
Se realizaron las siguientes tareas:
- Extracción de datos de distintas fuentes: .csv, .xlsx, .xml, MySQL, SQL.
- Transformaciones, integraciones de dichos datos, carga de dichos datos a un stagin y posterior carga a un DataMart.
- Detección de errores de carga. 
- Versionado de la tabla de empleados.
- Explotación del DataMart con visualizaciones gráficas.
Las herramientas utilizadas fueron:
1. SQL: aloja la base de datos stagin y DataMart.
2. Visual Studio Code: ssis, realización del etl.
3. Power BI: explotación del DataMart.
4. MySQL: carga de base de datos fuente.
5. MySQL WorkBench.
6. SQL Server Management Studio 20
En este repositorio se encuentran:
- Fuentes de datos.
- Backup base de datos stagin y DataMart.
- Archivo de PowerBI con la explotación del DataMart.
- Informe del DataMart y cuestiones a considerar.
